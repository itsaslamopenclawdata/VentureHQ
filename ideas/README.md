## How Ideas Are Evaluated

Each idea is scored across 5 dimensions:

1. **Problem Urgency** — How painful is the problem? (0-100)
2. **Market Potential** — TAM/SAM/SOM opportunity (0-100)
3. **Defensibility** — Can we build a moat? (0-100)
4. **Execution Feasibility** — Can we build it in 1 week? (0-100)
5. **Timing Readiness** — Is now the right time? (0-100)

Final score calculated by VenturescoreBot using weighted model:
- Problem: 20%
- Market: 20%
- Moat: 25%
- Execution: 20%
- Timing: 15%

## Decision Framework

| Score Range | Decision | Action |
|-------------|----------|--------|
| 80-100 | ✅ Strong Go | Invest immediately |
| 60-79 | ⚠️ Conditional Go | Investigate deeper |
| 40-59 | ❌ Likely No-Go | Major red flags |
| 0-39 | 🚫 Hard No | Don't pursue |

## Contributing Ideas

To submit an idea for evaluation:

1. Create a new folder with date: `ideas/YYYY-MM-DD/`
2. Add your idea with full analysis
3. Use format: `IDEAS_NAME.md`
4. Include all 5 dimension scores
5. Submit PR or mention @VenturescoreBot
